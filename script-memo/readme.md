## memo

+ Python v3でコンソールで対話するには `input`
    + `python3_input.py`

```
$ python3.7 python3_input.py
Plz tell me your first name
>> iganari
Hello !! iganari
```


+ Python v2でコンソールで対話するためには `row_input`
    + `python2_raw_input.py`

```
$ python2.7 python2_raw_input.py
Plz tell me your first name
>> iganari
Hello !! iganari
```
